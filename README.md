# 💈 Barbershop System Refatorado

## 📌 Status do Projeto
Em desenvolvimento 🚧

## 🛠 Tecnologias Utilizadas
- Java
- MySQL
- JDBC
- NetBeans

## 👨‍💻 Desenvolvedor
- Davi Santos

## 🎯 Objetivo do Sistema
Sistema para gerenciamento de uma barbearia, permitindo o controle de clientes, barbeiros, serviços e agendamentos.

## ⚙️ Funcionalidades
- Cadastro de clientes
- Cadastro de barbeiros
- Cadastro de serviços
- Realizar agendamentos
- Listar agendamentos
- Excluir agendamentos

## 🧱 Arquitetura
O sistema foi refatorado seguindo o padrão:
- Model (entidades)
- DAO (acesso ao banco)
- Service (regras de negócio)
- Controller (controle da aplicação)

Aplicando princípios SOLID, principalmente:
- SRP (Responsabilidade Única)
